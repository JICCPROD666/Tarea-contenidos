En esta pagina se expondran los atributos de etiquetas en html.

Primero revisaremos que es un atributo en html

  • Los atributos de HTML son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento. Los atributos de HTML son un modificador de un tipo de elemento de

Ahora sabiendo esto a continuacion una lista de atributos comunes en la programcion de html.

  1. Border
    • Enmarca cada celda de la tabla con un borde de grosor "n" representado en pixeles

  2. Align
    • se emplea para alinear una celda o tabla, tambien se puede usar para alinear un ideo, cancion o imagenes.

  3. Valign
    • se emplea para alinear verticalmente el contenido de una tabla o demas.

  4. Width
    • permite establecer el valor de ancho de una tabla

  5. Bgcolor
    • Se emplea para establecer un color de fondo

  6. Background
    • Se usa para establecer una imagen de fondo

  7. Cellspacing
    • Permite establecer el valor en pixeles de la separacion entre las celdas, de emplea en la etiqueta de table

  8. Colspan
    • Permite establecer una celda de abarque n columnas. Se emplea en la etiqueta de td

  9. Rowspan
    • Permite combinar una celda que abarque n filas, esta se coloca en la etiqueta de td

  10. Font-size
    • Permite cambiar el tamaño de la letra

  11. Color
    • Esta permite cambiar el color de la letra.

  12. Font-family
    • Esta permite cambiar la fuente del texto

  13. Text-align
    • Esta permite cambiar la alineacion del texto

  14. Ins
    • Esta permite subrayar el contenido encerrado en la etiqueta
